Algèbre des intervalles d’Allen¶

Pour faire des raisonnements sur des intervalles de temps, Allen a défini des relations possibles entre ces intervalles (Voir Algèbre des intervalles d’Allen).

Note

L’article de J.F. Allen, « Maintaining knowledge about temporal intervals », Communications of the ACM, No 11, Vol. 26, pp. 832-843, 1983, n’est pas en accès libre. Par contre le rapport dont il est issu est disponible sur le site de la bibliothèque de l’université de Rochester (Rapport Allen 1983).

On va construire une classe Intervalle comprenant plusieurs méthodes :

  • une méthode __init__(), constructeur de la classe. L’intervalle sera représenté par une liste à deux éléments ;

  • une méthode during() qui retourne True si l’intervalle considéré se déroule pendant l’intervalle passé en paramètre ;

  • une méthode __lt__() qui permet de surcharger l’opérateur < (relation before d’Allen) ;

  • une méthode __gt__() qui permet de surcharger l’opérateur > (inverse de la relation précédente) ;

  • une méthode __eq__() qui permet de surcharger l’opérateur == (relation equal d’Allen) ;

On ajoute à cette classe plusieurs méthodes qui ne font pas partie des relations d’Allen :

  • deux méthodes qui permettent de surcharger les opérateurs + et - avec les méthodes __add__ et __sub__ pour faire respectivement l’addition et la soustraction des intervalles ;

  • la méthode disjoint() qui retourne True si l’intervalle considéré est disjoint de l’intervalle passé en paramètre. On notera que cette méthode est traitée en utilisant les opérateurs >`et `< qui ont été surchargés à la place de conditions sur les bornes des intervalles ;

  • la méthode longueur() qui fournit la longueur de l’intervalle calculée comme la valeur absolue de la différence entre les deux bornes ;

  • la méthode __str__() pour l’affichage.

# -*- coding: utf-8 -*-

class Intervalle: 
    """
    Classe pour effectuer des calculs sur les intervalles
    """    
       
    def __init__(self, value):
        """
        Constructeur
        
        :param value: liste à deux éléments représentant repectivement les deux bornes de l'intervalle
        """
        self.value = value
           
    def during(self, other):
        """
        Méthode qui vérifie si l'intervalle self est contenu dans l'intervalle other passé en paramètre
        
        :param other: Intervalle
        :return: boolean
        """
        return other.value[0] < self.value[0] and other.value[1] > self.value[1]  
       
    def __lt__(self, other):
        """
        Méthode qui vérifie si l'intervalle self est avant l'intervalle other passé en paramètre
        
        :param other: Intervalle
        :return: boolean
        """        
        return self.value[1] < other.value[0]
           
    def __gt__(self, other):
        """
        Méthode qui vérifie si l'intervalle self est après l'intervalle other passé en paramètre
        
        :param other: Intervalle
        :return: boolean
        """           
        return self.value[0] > other.value[1]
       
    def __eq__(self, other):
        """
        Méthode qui vérifie si l'intervalle self est égal à l'intervalle other passé en paramètre
        
        :param other: Intervalle
        :return: boolean
        """              
        return self.value[0] == other.value[0] and self.value[1] == other.value[1]
    
    # Autres méthodes qui ne sont pas des relations de l'algèbre d'Allen    
    def disjoint(self, other):
        """
        Méthode qui vérifie si les intervalles self et other sont disjoints
        
        :param other: Intervalle
        :return: boolean
        """ 
        return self < other or self > other 
    
    def longueur(self):    
        """
        Méthode qui retourne la longueur d'un intervalle
        
        :return: longueur de l'intervalle 
        """
        return abs(self.value[0] - self.value[1])
       
    def __add__(self, other):
        """
        Méthode qui ajoute l'intervalle other à l'intervalle self
        
        :param other: Intervalle
        :return: Intervalle
        """          
        return Intervalle([self.value[0] + other.value[0], self.value[1] + other.value[1]])
           
    def __sub__(self, other):
        """
        Méthode qui retranche l'intervalle other à l'intervalle self
        
        :param other: Intervalle
        :return: Intervalle
        """            
        return Intervalle([self.value[0] - other.value[1], self.value[1] - other.value[0]])        
           
    def __str__(self):
        """
        Méthode pour l'affichage
        """
        return str(self.value)

Note

Attention à la soustraction des intervalles où les bornes du résultat ne sont pas la différence des bornes des opérandes.

Rectangle coloré défini par deux intervalles¶

On souhaite afficher un rectangle coloré défini par un intervalle qui représente sa position sur l’axe des abscisses, un intervalle qui représente sa position sur l’axe des ordonnées et une couleur. La couleur est représentée par une liste de 3 entiers entre 0 et 255 correspondant respectivement à l’intensité de rouge, de vert et de bleu (codage RVB). La figure suivante illustre ce principe.

Fig. 13 Définition d’un rectangle bleu par deux intervalles¶

Il existe plusieurs modules pour effectuer des dessins et gérer des interfaces avec l’utilisateur comme, par exemple, tkinter. Dans cet exemple, on a choisi d’utiliser pygame pour sa simplicité et pour la préparation des TP.

Important

le module pygame n’est pas encore disponible sur le canal par défaut d’Anaconda. On peut l’importer via la commande suivante dans une fenêtre de commande du système d’exploitation.

conda install -c cogsci pygame

On définit la classe Rectangle comme suit. Le constructeur de cette classe utilise deux intervalles, instances de la classe Intervalle ci-dessus.

# -*- coding: utf-8 -*-

import pygame
from ClasseIntervalle import Intervalle

class Rectangle:
    """
    Classe qui gère un rectangle coloré
    """
    
    def __init__(self, intervalleX, intervalleY, couleur):
        """
        Constructeur
        
        :param intervalleX: Intervalle sur l'axe des abscisses
        :param intervalleY: Intervalle sur l'axe des ordonnées
        :param couleur: liste de 3 entiers entre 0 et 255 définissant une couleur RVB
        """
        self.intervalleX = intervalleX
        self.intervalleY = intervalleY
        self.couleur = couleur
        # Initialisation de pygame
        self.initPygame()
        
    def initPygame(self):
        """
        Méthode qui initialise pygame
        """
        #Initialisation
        pygame.init()
        # Définition de la taille de l'écran.
        self.screen = pygame.display.set_mode([800, 600])                 

    def setCouleur(self, couleur):
        """
        Méthode qui permet de changer la couleur du rectangle (setter)  
        
        :param couleur: liste de 3 entiers entre 0 et 255 définissant une couleur RVB
        """
        self.couleur = couleur
    
    def coinHautGauche(self):
        """
        Méthode qui retourne le coin en haut à gauche du rectangle
        
        :return: liste contenant les coordonnées du coin en haut à gauche
        """
        return([self.intervalleX.value[0], self.intervalleY.value[0]])
        
    def taille(self):
        """
        Méthode qui fournit la taille du rectangle
        
        :return: liste contenant la largeur et la hauteur
        """
        return([self.intervalleX.longueur(),self.intervalleY.longueur()])  
        
    def position(self):  
       
        """
        Méthode qui fournit la position du rectangle sous la forme d'une 
        liste [haut, gauche, largeur, hauteur]
        
        :return: liste contenant la posiiton
        """      
        position = self.coinHautGauche()
        position.extend(self.taille())
        return position
        
    def surface(self):
        """
        Méthode qui fournit la surface du rectangle
        
        :return: surface
        """
        return(self.intervalleX.longueur() * self.intervalleY.longueur())         
    
    def dessine(self):
        """
        Méthode qui dessine le rectangle si pygame est initialisé
        """
        if pygame.display.get_init():
            pygame.draw.rect(self.screen, self.couleur, self.position())
            pygame.display.update()
            pygame.event.clear()
        
    def pause(self):
        """
        Méthode qui attend un événement Pygame
        """
        while True:
        # Waits for an event
            event = pygame.event.wait()
 
            if event.type == pygame.QUIT:
                self.quitte()                                  
            elif event.type == pygame.MOUSEBUTTONDOWN:
                return
            
    def quitte(self):
        """
        Méthode qui quitte Pygame
        """ 
        pygame.quit()           

if __name__ == '__main__':    
    X = Intervalle([12, 67])       
    Y = Intervalle([110, 250])
    couleur = [200, 0, 0]
    rectangle = Rectangle(X, Y, couleur)
    rectangle.dessine()
    rectangle.pause()
    rectangle.setCouleur([0,200,0])
    rectangle.dessine()
    rectangle.pause()   
    rectangle.quitte()

Rectangles colorés¶

On souhaite effectuer maintenant des traitements sur des rectangles colorés définis comme précédemment. Pour cela on crée une classe Rectangles qui contient une méthode ajoute() pour ajouter un rectangle coloré.

La classe contient également une méthode permettant d’encadrer le rectangle de plus grande surface en blanc par défaut avec une bordure d’un pixel par défaut.

# -*- coding: utf-8 -*-

import pygame
from ClasseIntervalle import Intervalle
from ClasseRectangle import Rectangle

class Rectangles:
    """
    Classe qui gère un rectangle coloré
    """
    
    def __init__(self):
        """
        Constructeur
        """
        self.rectangles = []

    def ajoute(self, rectangle):
        """
        Méthode qui permet d'ajouter un rectangle  
        
        :param rectangle: Rectangle à ajouter
        """
        self.rectangles.append(rectangle)
        
    def rectanglePlusGrandeSurface(self):
        """
        Méthode qui retourne le rectangle de plus grande surface
        """
        surfaceMax = 0
        resultat = None
        for rectangle in self.rectangles:
            surface = rectangle.surface()
            if surface > surfaceMax:
                surfaceMax = surface
                resultat = rectangle
        return resultat
    
    def encadreRectanglePlusGrandeSurface(self, couleur = [255, 255, 255], taille = 1):
        """
        Méthode qui encadre le rectangle de plus grande surface en blanc par défaut
        avec une bordure de 1 pixel par défaut
        """        
        rectangle = self.rectanglePlusGrandeSurface()
        pygame.draw.rect(rectangle.screen, couleur, rectangle.position(), taille)
        pygame.display.update()
        pygame.event.clear()  
          
    def dessine(self):
        """
        Méthode qui dessine tous les rectangles
        """
        for rectangle in self.rectangles:
            rectangle.dessine()
        
    def pause(self):
        """
        Méthode qui attend un événement Pygame
        """
        while True:
        # Waits for an event
            event = pygame.event.wait()
 
            if event.type == pygame.QUIT:
                self.quitte()                                  
            elif event.type == pygame.MOUSEBUTTONDOWN:
                return
            
    def quitte(self):
        """
        Méthode qui quitte Pygame
        """ 
        pygame.quit()           

if __name__ == '__main__':    
    r = Rectangles()      
    r.ajoute(Rectangle(Intervalle([12, 67]), Intervalle([110, 250]),[255, 0, 0]))
    r.ajoute(Rectangle(Intervalle([150, 160]), Intervalle([300, 400]),[0, 255, 0])) 
    r.ajoute(Rectangle(Intervalle([300, 400]), Intervalle([100, 200]),[128, 128, 0]))  
    r.dessine() 
    r.pause()
    r.encadreRectanglePlusGrandeSurface()    
    r.pause()    
    r.encadreRectanglePlusGrandeSurface([255, 0, 0], 3)
    r.pause()   
    r.quitte()
