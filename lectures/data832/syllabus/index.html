Modalités de cours

Ce cours est organisé en deux cycles qui abordent un aspect particulier du machine learning: l'apprentissage dit supervisé et l'apprentissage non supervisé.

Pour chaque partie, il y a 2 Cours Magistraux, 2 Travaux-dirigés et un TP sur 6 heures. Le TP séparé en 3 séances : 1 séances de lancement avec l'enseignant, 1 séance en autonomie, et une séance finale avec l'enseignant.

Modalités d'évaluation

L'évaluation se fait aux travers des deux TPs, qui sont des mini-projets, et au travers d'un examen terminal écrit avec le ratio 50% TPs et 50% Examen

Concernant les TPs, l'évaluation se fera à l'aide des éléments suivants :

  1. Un rendu de code python qui permet de résoudre le problème du projet et de faire une analyse des résultats. Concernant le format :
    • Rendu en archive zip;
    • Vous pouvez faire des notebook à condition de n'y mettre que l'exécution principale (toutes fonctions utiles devront être mise dans des fichiers pythons et importées);
    • Il est possible de ne faire que des fichiers python;
    • Un README devra expliquer les éléments pour reproduire vos résultats;
    • Lister toutes les dépendances dans le README avec la version, ou bien donner un fichier YAML pour conda.
  2. Un rendu de rapport avec les consignes suivantes :
    • 10 pages maximum ;
    • au format PDF;
    • vous devrez lister la problématique, expliquer quel type de problème d'apprentissage on affaire à, commenter le choix des algorithmes, expliquer les algorithmes employés ainsi que les méthodologies mises en place sur les données et donenr une analyse des résultats obtenus;
    • Imaginez le rapport comme un rapport technique pour votre futur vous pour vous rappeler comment refaire les tâches demandées ou bien comprendre les choses que l'on a obtenus en termes de résultat.
